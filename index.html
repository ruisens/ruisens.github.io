MAC平台最好用的播放器-IINA

发表于 2017-10-30 | 分类于 mac

从开始接触MACOS到入手了Macbook pro后一直在寻找一款播放器，试过不少主流的APP,但都没有觉得很满意，直到遇到了IINA，除了开源、美观、简介之外，可以打开各大主流视频网站url来观看（而且是免广告）

致力成为 macOS 最强的开源视频播放器

IINA 是一款免费开源的macOS万能视频播放器，基于mpv(一款命令行启动/高度自定义配置的高性能跨平台开源播放器) 而来，使用Swift语言开发，拥有强大的性能和兼容性，可以支持流畅播放几乎所有主流视频格式，并且支持网络播放。

IINA 视频播放器主界面截图

免费、开源

还有重要的一点是，IINA项目完全开源 (GPLv3)，而且作者也非常厚道地宣布：“IINA不管现在还是将来，都将免费”。因此，IINA极有可能在开源社区获得高速的成长，从而成为 Mac 平台上的播放器明日之星！

或许你也一样，一直期待着 macOS 上能一款美观、性能强大且能通吃所有视频的播放器出现。IINA 正是由有同样想法的用户开发的，并希望能同时满足普通和专业用户的媒体播放需求。如果你对目前使用的 Mac 视频播放器有各种不满，那么试试IINA吧，相信你不会失望的！

Github : https://lhc70000.github.io/iina/zh-cn/features/

谈谈MAC常用快捷键

发表于 2017-10-09 | 分类于 mac

身边不少朋友购买了macbook用来办公，购买初衷是喜欢macbook的轻薄、超长续航以及高颜值，入手后大部分都被软件下载、使用习惯所打败。macbook装windows系统，要么干脆直接换电脑。可能对于新手来说完全适应mac系统需要一段时间，不妨先从熟悉mac快捷键以及触摸板开始。

快捷键

Mac中主要有四个修饰键，分别是shift、control、option/alt、command。这4个键分别有自己的图案，他们经常出现在Mac应用程序中的菜单栏里，方便你随时学习新的快捷键。
这些总结的是非常常用的快捷键，如要更多快捷键请移步到官网：https://support.apple.com/zh-cn/HT201236
command是Mac里最重要的修饰键，在大多数情况下相当于Windows下的Ctrl。
